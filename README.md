# CHSportfolio



1. Gladiator Game (턴제 육성 스토리 RPG)



Turn-Based Strategy RPG Template 사용



- 구현 내용

1. Conversation2D 플러그인을 이용한 스토리 대화 및 스토리 엔딩 분기점 구현

- 스토리 대화 UI 구현

- 스토리 중 선택지를 통한 배드엔딩, 노말엔딩 분기점 구현

2. 육성 시스템의 N일차 시스템 및 스탯 시스템 구현

- 7일차 기준으로 주가 변하도록 구성

- 하루마다 훈련, 대화의 활동 중 하나를 선택하도록 함.

=> 선택시 화면 암전을 통해 하루가 넘어가도록 연출 추가 및 UI의 일차 추가

=> 7일차가 되면 전투맵으로 이동하여 전투 진행하도록 구성

=> 5주차가 되면 스토리 분기점이 나오며 엔딩 분기점을 선택하도록 함

3. 호감도 대화 시스템
