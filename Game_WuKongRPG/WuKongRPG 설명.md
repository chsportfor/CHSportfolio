# CHSportfolio

## 게임명 : 현천록 (소울라이크 액션 RPG)

### 프로젝트 개요
- **개발 기간:** 2025.9 ~ 2025.12
- **개발 인원:** 5명 (보스 전투 담당1, 아트및 이펙트 담당1, 몬스터 전투 담당1, 유저 애니메이션 담당1, UI, 게임 시스템 구현, 프로젝트 관리 1) /  UI, 게임 시스템 구현, 프로젝트 관리 담당
- **사용 기술 및 환경:** Unreal Engine 5, Blueprint 사용
- **장르:** 캐주얼 액션 RPG

### 주요 구현 내용
#### 1. 저스트 회피 시스템 구현
- **초산 공간 개념 차용:** "퍼니싱 그레이 레이븐"의 초산 공간(Matrix) 시스템을 참고하여, 저스트 회피 성공 시 적에게 슬로우 모션을 부여하고 화면 전체를 흑백으로 전환하는 포스트 프로세싱 효과를 연출했습니다.
- **전투 경험 강화:** 유저에게 정확한 타이밍의 회피를 통한 명확한 이점을 부여하여 액션성과 조작의 재미를 극대화했습니다.

**관련 Blueprints**
- **JustEvadeActivate:** 저스트 회피 판정 성공 시 트리거되는 커스텀 이벤트입니다.
- **시간 지연 및 시각 효과 제어:** `Set Global Time Dilation` 노드를 사용하여 게임 전체의 시간을 느리게(예: 0.2배속) 만들고, 플레이어 캐릭터의 `Custom Time Dilation`을 높여(예: 5.0배속) 플레이어만 정상 속도로 움직일 수 있도록 구현했습니다. 
- **포스트 프로세싱:** `Make Struct PostProcessSettings` 노드를 통해 화면의 `Saturation`(채도) 값을 0으로 낮춰 흑백 화면을 연출하고, 일정 시간(Delay)이 지나면 다시 원래의 시간 속도와 색상으로 복구하는 로직을 구성했습니다.

#### 2. 스태미나 시스템 구현
소울라이크 장르의 핵심인 자원 관리 요소를 위해 스태미나 시스템을 세밀하게 구현했습니다.
- **자원 소모 및 회복:** 회피, 달리기 등 특정 액션 수행 시 스태미나가 감소하며, 액션을 취하지 않을 때는 자연적으로 회복되도록 상태 기반 로직을 구성했습니다.
- **UI 보간(Interpolation) 연출:** 스태미나와 체력의 증감을 UI에 반영할 때, 수치가 즉각적으로 변하는 것이 아니라 `FInterp To` 노드를 사용하여 부드럽게 줄어들거나 차오르는 보간 이펙트를 구현했습니다. 이는 몬스터 헌터나 다크 소울 시리즈 등에서 볼 수 있는 묵직하고 시각적으로 인지하기 쉬운 UI 연출을 의도한 것입니다.

**관련 Blueprints**
- **스태미나 틱 업데이트 (Event Tick):** 매 프레임마다 플레이어의 상태(달리기, 회피 등)를 체크하여 스태미나 값을 증감시킵니다.
- **행동 제한 로직:** 스태미나가 0 이하로 떨어졌을 때 달리기, 점프 등의 특정 행동(`Can Dash`, `Can Jumping`, `Can Sprinting` 변수 제어)을 제한하는 `Branch` 로직을 구현하여 자원 관리의 중요성을 부각했습니다.
- **UI 보간 처리:** `Displayed Stamina` 및 `Displayed Hp` 변수를 별도로 두어, 실제 스탯 값(`Get Stamina`, `Get Hp`)과 목표 값 사이를 `FInterp To`로 부드럽게 연결하여 화면에 출력되도록 처리했습니다.

#### 3. UI 및 씬(Scene) 매니지먼트 구현
게임의 전반적인 흐름을 제어하는 다양한 UI 위젯과 시스템을 구축했습니다.
- **핵심 UI 위젯 제작:** 메인 메뉴(타이틀 화면), 로딩 화면, 일시 정지(Pause), 사망 화면(You Died), 엔딩 씬 등 게임 진행에 필요한 모든 주요 UI를 UMG를 통해 구현했습니다.
- **세이브 포인트(SavePoint) 시스템:** 맵 곳곳에 배치된 세이브 포인트 오브젝트와 상호작용 시, 플레이어의 체력을 회복(`Refill Stats`)하고 해당 위치를 부활 지점(`Respawn Location`)으로 GameInstance에 저장하는 로직을 구현했습니다. 상호작용 시 화면 중앙에 나타나는 세이브 완료 UI와 사운드 이펙트 또한 동기화하여 피드백을 강화했습니다.

#### 4. 프로젝트 및 버전 관리 (Microsoft Azure)
- **Azure DevOps 환경 구축:** 팀 프로젝트의 원활한 협업을 위해 Microsoft Azure 환경을 구축하고 소스 컨트롤(버전 관리)을 수행했습니다.
- **통합 및 충돌 해결 전담:** 여러 파트(아트, 기획, 타 개발자)의 작업물을 병합하는 과정에서 발생하는 블루프린트 충돌, 레벨 데이터 유실 등의 통합 이슈를 전담하여 처리했습니다. 문제 발생 시 신속한 버전 롤백 및 복구 작업을 통해 프로젝트 일정 지연을 방지했습니다.
